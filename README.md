# Ecommerce-Furniture-
This project demonstrates an end-to-end analytics workflow using Python, SQL, and Power BI to support data-driven business decisions.
Project Overview

This internship project analyzes e-commerce furniture sales data to understand sales performance, pricing behavior, shipping trends, and customer demand using Python, SQL, and Power BI.

 Objectives

Analyze total units sold and average product price

Classify products into price categories

Evaluate shipping type impact on sales

Identify top-selling furniture products

 Dataset

The project uses multiple datasets including total sales, price categories, shipping details, and top-selling products.
